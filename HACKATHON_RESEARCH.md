## 11. CODE RECIPES

### Browser Use SDK Setup (Python)
```bash
pip install browser-use-sdk
export BROWSER_USE_API_KEY=your_key
```

### Quickest Agent (Python v2)
```python
from browser_use_sdk import AsyncBrowserUse
from pydantic import BaseModel

class Result(BaseModel):
    title: str
    summary: str
    url: str

client = AsyncBrowserUse()
result = await client.run(
    "Go to news.ycombinator.com and get the #1 post",
    output_schema=Result,
    start_url="https://news.ycombinator.com"
)
```

### v3 Agent API
```python
from browser_use_sdk.v3 import AsyncBrowserUse

client = AsyncBrowserUse()
result = await client.run(
    "Find the top 3 trending repos on GitHub today",
    model="bu-mini",
    max_cost_usd=0.50,
    keep_alive=True
)
```

### Step Streaming
```python
async for step in client.run("Search google for 'browser use'"):
    print(f"Step {step.number}: {step.next_goal} — {step.url}")
```

### Skills
```python
skill = await client.skills.create(
    goal="Extract top X posts from HackerNews with title, URL, score, author",
    agent_prompt="Go to news.ycombinator.com, click first post, go back, scroll"
)
result = await client.skills.execute(skill.id, parameters={"X": 10})
```

### Convex Browser Use Component
```typescript
import { BrowserUse } from "browser-use-convex-component";
import { components } from "./_generated/api.js";
const browserUse = new BrowserUse(components.browserUse);

const task = await browserUse.createTask(ctx, {
  task: "Go to google.com and search for 'Convex agents'",
  startUrl: "https://google.com",
  maxSteps: 50,
  vision: true,
});
```

### Supermemory
```typescript
import Supermemory from "supermemory";
const memory = new Supermemory({ apiKey: process.env.SUPERMEMORY_API_KEY });

await memory.add({
  content: "User prefers dark mode and TypeScript",
  containerTags: ["user_123"],
});

const results = await memory.search.memories({
  q: "user preferences",
  containerTags: ["user_123"],
});
```

### AgentMail
```typescript
import { AgentMailClient } from "agentmail";
const mail = new AgentMailClient({ apiKey: process.env.AGENTMAIL_API_KEY });

const inbox = await mail.inboxes.create({ username: "my-agent" });
await mail.inboxes.messages.send(inbox.inbox_id, {
  to: ["recipient@example.com"],
  subject: "Hello from AI",
  text: "This email was sent by an AI agent.",
});
```
